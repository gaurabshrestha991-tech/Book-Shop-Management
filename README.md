# Book-Shop-Management
The Bookshop Management System is a simple C++ console application that helps manage a small book inventory. It allows users to add books, display all available books, and search for books by their title using a menu-driven interface.

---

## Features
- Add a new book
- Display all books in the inventory
- Search for a book by title
- Menu-driven interface
- Dynamic storage using vector

---

## Technologies Used
- C++
- Standard Template Library (STL)
  - vector
  - string
- Console Application

---

## Book Details Stored
Each book contains the following information:
- Title
- Author
- Quantity
- Price
